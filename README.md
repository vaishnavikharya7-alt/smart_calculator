# Smart Calculator

Advanced calculator with GUI interface supporting basic arithmetic, scientific functions, and history tracking. Built with Python and Tkinter.

## Features

- **Basic Operations**: Addition, subtraction, multiplication, division, percentages
- **Scientific Functions**: Trigonometric functions, logarithms, exponents, constants (π, e)
- **History Tracking**: Stores calculation history with timestamps
- **Keyboard Support**: Use keyboard for faster input
- **Modern GUI**: Clean interface with color-coded buttons
